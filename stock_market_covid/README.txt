This folder contains some lines of code that have been used to build a dataset with stock returns and 
company-level attributes to analyze the impact of the first wave of covid on the stock market.

All the information retrieved is publicly available, because I had hard times to get access
to market provided data when I was writing my Bachelor Thesis in the midst of the first wave of Covid-19.

Project: Bachelor Thesis.
Supervisor: Professor Giuseppe Ragusa
Software: R
